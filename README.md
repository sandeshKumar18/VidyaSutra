# VidyaSetu

**VidyaSetu** is a modern AI-powered learning platform built using the MERN stack that helps learners discover, organize, and follow personalized learning roadmaps. The platform combines intelligent curriculum generation, user management, and an engaging user experience to make skill development more structured and accessible.

## 🚀 Features

* 🔐 Secure User Authentication & Authorization
* 🤖 AI-Powered Curriculum Generator
* 📚 Personalized Learning Roadmaps
* 👤 User Profile & Progress Tracking
* 🎨 Modern Responsive UI with Tailwind CSS
* ⚡ Fast and Interactive User Experience
* 📱 Mobile-Friendly Design
* 🔍 Resource Discovery and Learning Management
* 🌙 Clean Dashboard Interface
* 📊 Structured Learning Path Visualization

## ⚙️ Installation

### Clone Repository

```bash
git clone <repository-url>
cd VidyaSetu
```

### Install Frontend Dependencies

```bash
cd client
npm install
```

### Install Backend Dependencies

```bash
cd ../server
npm install
```

### Configure Environment Variables

Create a `.env` file inside the server directory:

```env
PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
AI_API_KEY=your_ai_api_key
```

### Start Development Server

Backend:

```bash
npm run dev
```

Frontend:

```bash
npm run dev
```
